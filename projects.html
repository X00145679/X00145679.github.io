Online Whiskey Store

Description

2nd year team project involving the development of an online store where multiple users can bid on products and list products for auction. Some of the features include account authorization and a database with create update edit and delete functionality and a shopping cart to store all items bid on or purchased and the storing of previously purchased products for users to view.
Challenges Faced:
• Implementation of the database and the adding of auction functionality
• Increasing security to prevent users data from being stolen

Tools & Technology Used:
PlayFramework, Java , Javascript , HTML5 , CSS , SQL
